Popover CSS Inspector

Automatically display CSS properties for an element - perfect for design systems and style guides

  • Easy to use - just add data-bs-custom-class="css-inspector" to the element (requires Bootstrap 5 JS)
  • Customizable with your own CSS and Bootstrap's Popover options
  • Option to hide CSS elements - e.g. just add data-css-inspector="hide-fs" to prevent font-size from displaying
  • Some CSS properties are hidden by default - e.g. just add data-css-inspector="show-b" to show border properties
  • Only 2 KB minified and gzipped!
